## How to Run (Testing)

1.  **Starting the Project:**
    Download the project to your computer using `git clone`. Then, open the Unreal Engine editor by clicking on the **`CarpenterGame.uproject`** file located in the `CarpenterGame` folder.

2.  **Multiplayer Settings:**
    From the options next to the Play button, set the **Number of Players** to **4**, and play with Listen Server Mode.


---

## Controls

The following key bindings are used for in-game interactions:

| Key | Action | Description |
| :---: | :--- | :--- |
| **WASD** | **Walk** | Walk around |
| **E** | **Pickup & Drop** | Pickup, drop and interact tables while you have an item |
